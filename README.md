## Quick start

1.  Make sure that you have Node.js v10.13.0 and npm v6.4.1 or above installed.
2.  Clone this repo using `git clone https://github.com/gvete/itunes-top-app.git`
3.  Run `npm install` or `yarn install` or `yarn` command.
4.  Run `npm run start` or `yarn start` to run app at `http://localhost:3000`.
5.  Open `http://localhost:3000` in your browser.

I have completed almost all UI as per given in the assignment. Also written few Test cases for following modules:
1. app/components/Footer
2. app/store
